# Симуляция пружинного маятника (Unity)

Интерактивная модель затухающих колебаний системы «груз на пружине». Движение рассчитывается в реальном времени с учётом силы упругости и силы сопротивления.

## Возможности
- Запуск и сброс симуляции
- Настройка физических параметров: масса, жёсткость пружины, коэффициент затухания, начальное смещение
- Визуализация движения груза и процедурная отрисовка пружины
- Отображение текущего состояния, времени моделирования, текущего и максимального смещения

## Физическая модель
Движение описывается уравнением:
`F = -k·x - c·v`

Где:
- k — жёсткость пружины
- x — смещение от положения равновесия
- c — коэффициент затухания
- v — скорость груза

## Технологии и структура проекта
Стек: Unity, C#, TextMeshPro.

Основные скрипты:
- SpringMassController.cs — расчёт физики движения
- SpringLineRenderer.cs — визуализация пружины
- UIManager.cs — управление интерфейсом и вывод параметров

## Как запустить
1. Открыть проект в Unity Hub
2. Загрузить сцену проекта
3. Нажать Play
4. Ввести параметры и нажать Старт
